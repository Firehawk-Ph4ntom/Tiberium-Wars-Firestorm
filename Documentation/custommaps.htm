Custom Maps

Basic Explanation

With the updated version of Worldbuilder it is possible to build maps which use new assets added with your mod. To do so you need to edit the configuration file for Worldbuilder to load your mod. After the mod is created, there is nothing else you need to do, simply launch your mod in C&C 3 and play on your new map.

Configuring Worldbuilder

To launch Worldbuilder with a mod, you have to edit the CNC3_wb_1.0.cfg file, located in the root of your C&C 3 directory. You will need to add a line similar to the following to the top of the file:

try-add-config C:\Documents and Settings\<your username>\My Documents\Command & Conquer 3 Tiberium Wars\mods\SampleMod\SampleMod_1.0_SkuDef

This will tell Worldbuilder to attempt to load your mod, and allow you to create maps using the new assets added in your mod.

Custom Maps and Overrides

If you have created map that uses an override you will need to build this map using the BinaryAssetBuilder.  We have supplied a batch file to do this for you.  Simply run the BuildMap.bat from the root of the MOD SDK install directory passing it the name of your map.  For example:

BuildMap.bat SampleMap

This file will copy your map to a location so it can be built then copy the output files into your map directory.  This is required when using overrides.
